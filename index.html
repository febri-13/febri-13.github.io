XAMPP Installation Guide

Welcome! This guide will teach you how to install XAMPP, a tool that helps create websites on your computer before going live on the internet.

Step 1: Downloading XAMPP

  1. Open a browser (like Chrome or Firefox).
  2. Type https://www.apachefriends.org in the address bar and press Enter.
  3. Select the XAMPP version that matches your operating system (Windows, macOS, or Linux), and click Download.

Step 2: Starting the Installer

  1. Go to your Downloads folder and double-click the downloaded XAMPP file.
  2. If prompted with “Do you want to allow this app to make changes to your device?”, click Yes.
  3. The installation window will appear; click Next to continue.

Step 3: Choosing Components

Select only the necessary components:

  • Apache: This allows your computer to act as a server for websites.
  • MySQL: Used to store website data like user info.
  • PHP and phpMyAdmin: These support working with databases.

Click Next when done.

Step 4: Choosing Installation Location

Choose a location for installing XAMPP:

  • It will suggest a default location, typically C:\xampp.
  • Click Next to continue with the suggested location.

Step 5: Starting the Installation Process

  1. Review the selected options and click Next to start installation.
  2. Wait until the installation completes (may take a few minutes).

Step 6: Opening XAMPP Control Panel

  1. Once installed, check the box to launch the XAMPP Control Panel and click Finish.
  2. The Control Panel lets you start and stop XAMPP components like Apache and MySQL.

Step 7: Starting Apache and MySQL

  1. In the Control Panel, click Start next to both Apache and MySQL.
  2. If both start successfully, they will show green indicators next to them.

Step 8: Testing the XAMPP Installation

  1. Open a browser and type http://localhost/ in the address bar, then press Enter.
  2. If the XAMPP welcome page appears, Apache is running correctly.
  3. For MySQL, type http://localhost/phpmyadmin/ and press Enter. You should see the phpMyAdmin interface.

Conclusion

Congratulations! You've successfully installed XAMPP, which will allow you to develop websites locally on your computer.

In future lessons, we’ll explore how to use XAMPP with tools like WordPress to make building websites easier.
